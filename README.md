# ConnectNow - Real-Time Chat Application

ConnectNow is a real-time chat application built using React, Node.js, Express, Socket.IO, and PostgreSQL.

## Features

- User Registration & Login
- Real-Time Messaging
- Multiple Chat Rooms
- Create New Rooms
- Online User Status
- Image Upload Support
- Document Upload Support
- Emoji Support
- Responsive UI

## Tech Stack

### Frontend
- React.js
- Axios
- Socket.IO Client
- CSS

### Backend
- Node.js
- Express.js
- Socket.IO

### Database
- PostgreSQL

### Deployment
- Frontend: Vercel
- Backend: Render
